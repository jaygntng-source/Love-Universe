# OUR LITTLE UNIVERSE

## A Digital Love Story Between Fikri & Zahra

Premium Interactive Romantic Digital Experience

---

### 📖 Project Description

OUR LITTLE UNIVERSE is a premium digital love story website that captures the journey of Fikri and Zahra, a relationship that began unexpectedly through Mobile Legends and grew into a beautiful long-distance connection.

The website serves as a "digital memory capsule" or "digital love museum" — elegant, modern, and alive with cinematic storytelling.

---

### ✨ Features

- **Cinematic Opening** — Immersive landing experience
- **Relationship Counter** — Real-time day/month/year counter
- **Story Timeline** — Interactive journey through key moments
- **Memory Gallery** — Pinterest-style photo gallery with lightbox
- **Love Letter** — Animated letter with glass effect
- **Secret Box** — Hidden message with interactive reveal
- **Gift Card** — Sparkle animation gift
- **Food Favorites** — Display favorite foods of each person
- **Someday We Meet** — Hope list for future meeting
- **3D Heart Animation** — Cinematic final ending
- **Music Player** — iOS-style player with progress bar
- **Cinema Mode** — Immersive dark mode experience
- **Custom Cursor** — Premium glowing dot cursor (desktop)
- **Responsive Design** — Mobile-first approach
- **Easter Egg** — Hidden sparkle effect
